# 🧠 Portafolio de Agentes con LangChain y LangGraph

Este repositorio recopila proyectos que muestran cómo construir agentes inteligentes utilizando LangChain y LangGraph. El objetivo es demostrar mi dominio en el diseño de flujos conversacionales, integración con modelos de lenguaje (LLMs), herramientas externas y manejo de estado en flujos de IA.

---

## 🚀 Proyectos Incluidos

| Proyecto | Descripción | Tecnologías |
|----------|-------------|-------------|
| `weather_agent.ipynb` | Agente que extrae una ciudad desde el mensaje del usuario y consulta el clima actual con OpenWeatherMap | LangGraph, LangChain, Gemini/OpenAI |
| `tool_router.ipynb` | Agente que enruta preguntas del usuario hacia distintas herramientas según su intención | LangChain, ToolRouter, LLMs |
| `multi_step_graph.ipynb` | Grafo condicional que toma decisiones basadas en el estado del flujo (como seguir o detenerse) | LangGraph, Condicionales, States |

---

## 🧰 Tecnologías Usadas

- 🧠 [LangChain](https://www.langchain.com/)
- 🔁 [LangGraph](https://docs.langchain.com/langgraph/)
- 🤖 Modelos LLM (Gemini, OpenAI, etc.)
- 🔧 Herramientas personalizadas (API externas, cálculos, etc.)
- 📦 Poetry (para gestión de dependencias)
- 📝 Jupyter Notebooks
